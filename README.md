🎓 CollegeDocs - Automated Document Generation System 

CollegeDocs is a full-stack web application developed to automate and simplify the college document request and approval process. The system allows students to request certificates online, track their status in real time, and receive approved documents digitally without visiting administrative offices.

⚙️ Features

👨‍🎓 Student Module

* Student Registration & Login
* Secure OTP-based Authentication
* Apply for Document Requests
* Track Request Status in Real Time
* Download Approved Certificates
* Email Notifications for Request Updates

👨‍🏫 Faculty Module

* Class Incharge Dashboard
* HOD Dashboard
* Request Approval / Rejection Workflow
* Branch & Section Based Request Filtering
* Real-Time Request Management

👨‍💼 Admin Module

* Faculty Management
* Student Management
* System Statistics Dashboard
* User Monitoring & Administration

📄 Certificate Generation

* Automatic PDF Certificate Generation
* Unique Certificate ID
* QR Code Verification
* Principal Signature Integration
* Downloadable Digital Certificates

📧 Email Automation

* OTP Verification Emails
* Approval / Rejection Notifications
* Certificate Delivery Emails
* Brevo Email Integration

⸻

🏗️ System Workflow

1. Student submits a document request.
2. Request is forwarded to the respective Class Incharge.
3. After approval, the request moves to the HOD.
4. HOD approves the request.
5. Certificate PDF is automatically generated.
6. Student receives an email notification and can download the certificate.
